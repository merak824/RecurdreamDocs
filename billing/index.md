# 充值说明

Recurdream 使用余额或套餐方式控制可用额度。实际展示以站点页面为准。

## 充值流程

1. 登录 Recurdream。
2. 打开 **充值/订阅**。
3. 选择快捷金额或输入自定义金额。
4. 选择支付方式。
5. 完成支付后返回站点查看余额。

## 到账规则

支付成功后，系统会自动将金额充值到你的账户余额。若短时间内没有到账：

1. 刷新页面。
2. 查看 **我的订单**。
3. 确认支付平台是否显示成功。
4. 仍未到账时联系站点客服，并提供订单号。

## 余额消耗

余额会根据模型、输入输出 Token、缓存、图片或其他计费项扣减。不同模型单价不同，具体以站点页面展示为准。

## 订单状态

| 状态 | 含义 |
| --- | --- |
| 待支付 | 已创建订单，尚未完成支付 |
| 已支付 | 支付平台已确认，等待系统入账 |
| 已完成 | 余额已到账 |
| 已过期 | 长时间未支付，订单关闭 |
| 失败 | 支付或入账失败，需要重试或联系客服 |

## 退款

退款规则由站点运营方决定。若需要退款，请联系站点客服并提供账号、订单号、支付时间和支付金额。
