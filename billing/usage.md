# 余额与用量

你可以在控制台查看余额、请求记录和 API Key 维度的消耗情况。

## 查看余额

登录后在仪表盘或个人账户页面查看当前余额。

如果余额不足，客户端请求可能返回余额不足、403 或 402 类错误，具体取决于客户端显示方式。

## 查看用量

打开 **使用记录** 可以查看：

- 请求时间
- 使用的 API Key
- 模型名
- 输入和输出 Token
- 费用
- 请求状态

## API Key 额度

创建或编辑 API Key 时，可以设置额度上限。适合下面场景：

- 给测试客户端设置小额度，避免误用。
- 给团队成员单独分配预算。
- 给不常用客户端设置有效期。

## 用量异常怎么办

如果发现异常消耗：

1. 先禁用对应 API Key。
2. 查看使用记录里的模型、时间和 IP。
3. 创建新 Key，并只配置到可信客户端。
4. 如有必要，联系站点客服协助排查。
